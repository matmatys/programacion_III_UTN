# c13-chat-app-end

Final snapshot for Class 13 (frontend-backend integration).

Includes:
- HTTP integration using `provideHttpClient`,
- shared response contracts in `core/basic`,
- API layer (`base-api`, `auth-api`, `chat-api`),
- login integration with `POST /auth/login`,
- conversation/message integration with backend endpoints,
- conversation actions: create, activate, archive.

## Run

```bash
npm install
npm run start
```

Open: `http://localhost:5300`
